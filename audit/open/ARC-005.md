# Architecture Finding: ARC-005

## Missing Observability Layer

**Severity:** MEDIUM  
**Impact:** No visibility into production issues  
**Status:** 🔴 **OPEN**

---

## Description

Metrics collected but not exported. No tracing or structured logging.

### Current State

```v
pub fn (s &MetricsService) get_all_metrics() map[string]MetricData {
    // Internal only - no export
}

// No tracing
// No structured logging
// No health endpoint
```

---

## Remediation

Add observability stack:

```v
pub interface IObservability {
    mut:
        trace(span Span)
        metric(name string, value f64)
        log(entry LogEntry)
}

// Export to Prometheus
pub struct PrometheusExporter {
    port int
}

pub fn (mut e PrometheusExporter) scrape() string {
    mut output := ''
    for name, value in metrics {
        output += '# TYPE ${name} gauge\n'
        output += '${name} ${value}\n'
    }
    return output
}

// Export to Jaeger/Zipkin
pub struct JaegerTracer {
    agent_host string
    agent_port int
}

// Structured logging
pub fn log_json(entry LogEntry) string {
    return json.encode({
        'timestamp': entry.timestamp
        'level': entry.level
        'message': entry.message
        'context': entry.context
    })
}
```
